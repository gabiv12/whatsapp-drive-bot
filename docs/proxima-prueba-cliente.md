# Próxima Prueba Con Cliente

## Antes De Que Venga

- Verificar que Render esté online.
- Verificar que Twilio Sandbox apunte a Render.
- Verificar variables de entorno en Render.
- Tener lista la cuenta Google del cliente.
- Tener instalado Google Drive para escritorio si se puede.

## Cuando Venga

- Unir su WhatsApp al Sandbox con `join ...`.
- Enviar `hola`.
- Enviar PDF con `pablo C001`.
- Enviar PDF con `fernando C001`.
- Confirmar respuesta del bot dentro de WhatsApp.
- Configurar Drive si corresponde.
- Probar subida real cuando esté implementada.

## Pendiente Técnico

- Descargar archivo real desde Twilio MediaUrl.
- Subir archivo real a Drive.
- Sincronizar en PC con Google Drive para escritorio.
